# COSC 499 Group Exercise (Sorting Integer Arrays)

> Authors:  Qingyan Hu, Fareeha Hayat, Orvin Tritama, Ryan Soon Han Lu\
> Project Date: 2022W Week 3

### Description
This repository is created for COSC499 Group GitHub exercise. In this exercise, our team members will implement a Java program with four independent features that is based off sorting integer arrays. Each of our group members has chosen a distictive sorting method that is often used for sorting integers. The objective of this team exercise is to become familiar with using git in a software development environment as well as getting used to generating GitStats results for evaluation.

### Features
1. **Selection Sort**
2. **Insertion Sort**
3. **Bubble Sort**
4. **Bucket Sort**

### Structure
All features will be condensed into one class file including the four features presented as methods. The project will also include a test file that only includes test cases for unit testing purposes. The unit testing will be performed using JUnit 4.

### Evaluation
The evaluation of our personal efforts will be performed using GitStats. Each of our team members will submit a GitStat report to Canvas.
